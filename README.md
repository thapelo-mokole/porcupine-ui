## Getting Started

* [Download the installer](https://nodejs.org/) for Node LTS version `v18.17.0`
* Install the ionic CLI globally: `npm install -g ionic`
* Clone this repository: `git clone https://github.com/thapelo-mokole/porcupine-ui.git`.
* Run `npm install` from the project root.
* Run `ionic serve` in a terminal from the project root.
